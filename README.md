# AI-Powered IT Support System

A full-stack web application designed to help users submit and track IT support tickets, with AI-generated troubleshooting suggestions.

## Features

- Submit IT support tickets
- View submitted tickets
- Categorize issues by type and priority
- Generate AI-based troubleshooting suggestions
- Store ticket data using SQLite
- Backend API built with Node.js and Express

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: SQLite

## Project Structure

- `frontend/` – user interface files
- `backend/` – server, routes, and database logic
